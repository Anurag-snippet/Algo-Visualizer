# 🌌 Algo Visualizer Hub

An interactive, premium, and responsive web application built to visualize sorting and pathfinding algorithms in real-time. Designed with a sleek modern interface, customizable controls, and step-by-step execution.

🔗 **Live Deployment:** [https://Anurag-snippet.github.io/Algo-Visualizer/](https://Anurag-snippet.github.io/Algo-Visualizer/)

---

## ✨ Features

### 📊 1. Sorting Visualizer
*   **Supported Algorithms:** Bubble Sort, Quick Sort, Merge Sort.
*   **Array Configurations:** Random, Reversed, and Nearly Sorted arrays.
*   **Controls:** Adjustable array size, execution speed (0.5x to 4x), play/pause, step-by-step navigation, and reset.
*   **Visual States:** Clear color legends for Unsorted, Comparing, Swapping, and Sorted elements.

### 🕸️ 2. Pathfinding Visualizer
*   **Supported Algorithms:** Dijkstra's Algorithm (and more).
*   **Interactive Grid:** Draw custom obstacles/walls and watch the search path expand dynamically to find the shortest path from start to target.

### ⚡ 3. Architecture & Engine
*   **Framework:** Built on **TanStack Start** (React + Vite + Nitro).
*   **Routing:** File-based client-side routing.
*   **Static Prerendering (SSG):** Configured to automatically generate static HTML pages for high-performance and seamless static hosting compatibility.

---

## 🛠️ Technology Stack

*   **Frontend Library:** React 19
*   **Routing & Framework:** `@tanstack/react-start`
*   **Styling:** Tailwind CSS (v4)
*   **Icons:** Lucide React
*   **Bundler:** Vite 7
*   **Hosting:** GitHub Pages (prerendered SSG) / Vercel compatible

## 🚀 Getting Started

### Prerequisites
Make sure you have Node.js installed on your machine.

### Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/Anurag-snippet/Algo-Visualizer.git
   cd Algo-Visualizer
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the local development server:
```bash
npm run dev
```

### Production Build
Build the project for production:
```bash
npm run build
```
